##  Project Overview

Address Book is a modular, menu-driven console application developed in C that enables users to manage contacts efficiently.  
The system supports full CRUD operations (Create, Read, Update, Delete) along with input validation and persistent storage using file handling.
The application loads existing contacts at startup and saves them to a structured text file, ensuring data persistence across executions.

##  Features

- Create new contact
  Search contact by:
- Name
- Phone number
- Email
- Edit existing contact details
- Delete contact
- List all contacts in formatted table view
- Save contacts to file
- Auto-load contacts at startup
- Phone number validation (10 digits, unique)
- Email validation (.com format, unique)
- Partial search support using `strstr()`
